# User Story: Xem bảng tính Excel (.xlsx)

**As a** Người dùng DocsViewer,
**I want to** xem file `.xlsx` dưới dạng grid và chuyển giữa các sheet,
**So that** tôi đọc được dữ liệu bảng tính trên web mà không cần cài Microsoft Excel.

## Acceptance Criteria

```gherkin
Scenario: Hiển thị sheet dạng grid
  Given tôi đã tải lên một file .xlsx hợp lệ
  When hệ thống render tài liệu
  Then sheet hiện tại được hiển thị dạng grid với giá trị từng ô
  And công thức được hiển thị ở dạng kết quả tính toán

Scenario: Chuyển giữa nhiều sheet
  Given tôi đang xem một .xlsx có nhiều sheet
  When tôi chọn một sheet khác
  Then hệ thống hiển thị nội dung của sheet được chọn (UC-02 A1)

Scenario: Trang đầu hiển thị trong ngưỡng thời gian
  Given tôi tải lên một .xlsx kích thước trung bình
  When hệ thống render sheet đầu tiên
  Then thời gian mở & hiển thị trang đầu ≤ 3 giây (baseline NFR-01 §4.2, KR1.3)
```

## Unhappy Path

```gherkin
Scenario: .xlsx hỏng / không parse được
  Given tôi tải lên một file .xlsx có nội dung hỏng
  When hệ thống cố render
  Then hệ thống hiển thị thông báo lỗi và không render nội dung sai

Scenario: File quá lớn vượt giới hạn trình duyệt
  Given tôi tải lên một .xlsx rất lớn vượt ngưỡng `MAX_FILE_SIZE` (NFR-07 §4.1)
  When hệ thống cố nạp & render
  Then hệ thống từ chối hoặc cảnh báo giới hạn thay vì làm treo trình duyệt (Risk R-05)
```
