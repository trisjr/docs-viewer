## 3. Thứ tự build đề xuất (dependency-aware)

```
ST-01 Upload  →  ST-02 Layered Processing  →  ST-03/04/05 View (PDF/.docx/.xlsx)
   →  ST-06 Unified Viewer  →  ST-07/08/09 Extract  →  ST-10 Copy/Export
   →  ST-11 Search  →  ST-12 Highlight & Navigate
```

- Upload + Layered Processing là nền (mọi thứ phụ thuộc).
- Extract phụ thuộc View đã mở tài liệu; Search phụ thuộc Extract đã có nội dung.
- ST-10 & ST-12 (Should) có thể trượt cuối M1 nếu cần.
