# Epic: Multi-Format Viewing

## Mục tiêu (Goal)

Xem được cả 3 định dạng lõi (`.docx`, `.xlsx`, PDF) trên web ở mức **Acceptable Fidelity**, trong một giao diện xem thống nhất (Unified Viewer) — không cần cài phần mềm.

## Phạm vi (Scope)

- Render PDF: phân trang, zoom cơ bản.
- Render `.docx`: text, heading, bảng, hình cơ bản ở Acceptable Fidelity.
- Render `.xlsx`: hiển thị sheet dạng grid (giá trị ô; công thức hiển thị kết quả); chuyển sheet.
- Unified Viewer UI: một UI chung tự nhận diện định dạng & chọn viewer phù hợp; điều hướng nhất quán.
- Out of scope (defer M4): `.pptx` và định dạng khác; edit nội dung.
