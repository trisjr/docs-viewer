# Epic: Platform Foundation

## Mục tiêu (Goal)

Đưa tài liệu vào hệ thống (Ingestion/Upload) và đặt nền kiến trúc **layered** sẵn sàng mở rộng — tách lớp `Document Processing` khỏi UI và tầng user để về sau thêm định dạng mới hoặc gắn auth/multi-tenant mà không phải viết lại core.

## Phạm vi (Scope)

- Chọn/kéo-thả 1 file; validate định dạng (`.docx/.xlsx/pdf`) + ngưỡng size; từ chối file sai/quá lớn/hỏng với thông báo rõ ràng.
- Tách lớp logic `parse/extract` (enabler) độc lập với UI; cung cấp extension point cho định dạng mới và auth/multi-tenant về sau.
- Out of scope (defer M3): Upload nhiều file/hàng loạt, lưu trữ tài liệu lâu dài, auth/multi-tenant đầy đủ.
