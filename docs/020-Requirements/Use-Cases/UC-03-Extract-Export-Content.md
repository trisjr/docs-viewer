# Use Case 03: Trích xuất & Sao chép/Export nội dung

> Phủ MVP-Scope item 4: bóc tách text từ `.docx`/PDF, data dạng bảng từ `.xlsx`, và cho phép sao chép/export **nội dung đã trích xuất**.

| Thuộc tính        | Mô tả                                                                                                           |
| :---------------- | :-------------------------------------------------------------------------------------------------------------- |
| **Actor**         | Người dùng (User) — người cần lấy nội dung tài liệu để search/feed AI/dùng nơi khác.                             |
| **Precondition**  | Một tài liệu đã được tải lên và đang hiển thị (xem [UC-02](./UC-02-Upload-View-Document.md)).                    |
| **Trigger**       | User chọn trích xuất (extract) nội dung của tài liệu đang xem.                                                   |
| **Postcondition** | Nội dung text/data được trích xuất và hiển thị; User có thể sao chép (clipboard) hoặc export ra file text/data.  |

## Main Flow

1. User đang xem một tài liệu (PDF/`.docx`/`.xlsx`).
2. User chọn thao tác trích xuất nội dung.
3. Hệ thống bóc tách nội dung theo loại tài liệu:
   - PDF / `.docx`: trích xuất text.
   - `.xlsx`: trích xuất data dạng bảng (giá trị ô theo hàng/cột).
4. Hệ thống hiển thị nội dung đã trích xuất cho User xem lại.
5. User chọn sao chép vào clipboard hoặc export nội dung đã trích xuất ra file (text cho PDF/`.docx`, data dạng bảng cho `.xlsx`).
6. Hệ thống thực hiện sao chép/tạo file tải về và xác nhận thành công.

> [!NOTE]
> "Export" ở đây = xuất **nội dung đã trích xuất** (text/data). Convert tài liệu sang định dạng khác (vd Word→PDF, Excel→JSON): **Out of scope — defer Full Scope.**

## Alternative Flows

- **A1 — Chỉ sao chép vào clipboard:** Tại bước 5, User chọn copy thay vì export file; hệ thống ghi nội dung đã trích xuất vào clipboard và xác nhận. Use case kết thúc.
- **A2 — Trích xuất một phần (`.xlsx` theo sheet):** Tại bước 3, với `.xlsx` nhiều sheet, User chọn sheet cần trích xuất; hệ thống chỉ bóc tách data của sheet đó rồi tiếp tục bước 4.

## Exception Flows

- **E1 — PDF dạng scan/ảnh, không có text:** Tại bước 3, nếu PDF là ảnh scan không chứa text layer, hệ thống không trích xuất được text và thông báo cho User. Use case kết thúc. _(OCR cho PDF scan: Out of scope — defer Full Scope.)_
- **E2 — Nội dung trích xuất rỗng:** Tại bước 4, nếu tài liệu không có nội dung text/data (vd file trống), hệ thống thông báo không có nội dung để sao chép/export. Use case kết thúc.
- **E3 — Lỗi parse khi trích xuất:** Tại bước 3, nếu việc bóc tách thất bại do nội dung hỏng, hệ thống hiển thị thông báo lỗi và không tạo kết quả. Use case kết thúc.
